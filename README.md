# fw-theme-smart
A theme for ForceWebsite


# Config
Add this to config file for theme section
```json
{
  "theme": {
    "namespace": "smart",
    "host": "https://9r3i.github.io/fw-theme-smart",
    "config": {
      "data": {
        "limit": 8
      }
    }
  }
}
```
